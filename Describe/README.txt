Plugin to remember character descriptions by inserting them into a mysql database.
This is for future possible web addition of character management.

To set description:
!setdesc
Example: !setdesc writes sexy python code.

It can also set links to wiki or image links, they will be returned with !describe:
!setlink <url>

To recall description and links:
!describe <character>

Example:
[IRC] David writes sexy python code
[IRC] Link: <url>
